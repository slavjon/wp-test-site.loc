На основе CMS Wordpress и пустой темы(например underscores.me) реализовать следующий функционал: Используя плагин ACF: Добавить возможность загружать лого в шапку(option page) а также номер телефона и емейл Продукты на главной странице должны задаваться при помощь ACF(Custom page template) Меню в шапке должно редактироваться из админ панели Используя Woocommerce: Сделать вывод карточек товаров в слайдере на главной странице(при необходимости создать несколько тестовых товаров) Вывод рейтинга(звёздочек) должен происходить на основе отзывов о товаре(шаг округления 0.5) Предусмотреть вывод вариативных товаров. Имеется ввиду что если для товара задано несколько цен, то вывести наименьшую Требования к верстке: Слайдер Slick Использование Scss и методологии БЭМ Наличие адаптивной верстки до 320px(реализация на ваше усмотрение) Сборщик написать на gulp(хотя бы для Scss) Дополнительное задание: Реализовать мультиязычность(RU/EN) используя один из плагинов Реализовать кнопку лайк на карточке товара(добавить в избранное)
